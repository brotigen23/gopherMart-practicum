# Гофмаркет
## Запуск
### Создание Docker образов
```
make init
```
### Запуск Docker compose
```
make run
```
## Описание
Данный проект представляет собой работу сервиса, реализующего систему лояльности по накоплению баллов за покупки  пользователем.
